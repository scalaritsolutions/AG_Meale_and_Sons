# A.G. Meale & Sons | Official Website

![A.G. Meale & Sons](public/favicon.png)

A high-fidelity, modern, and mobile-responsive digital experience for A.G. Meale & Sons—a family-run farm shop, nurseries, and plant centre in Stalham, Norfolk.

## 🌟 Key Features

- **Full Navigation Parity**: 17+ pages covering everything from the Farm Shop and Vera's Coffee Shop to Seasonal Trees and the Maize Maze.
- **Premium User Experience**: 
  - **Mega-Menu**: Categorized desktop navigation for effortless browsing.
  - **Mobile-First Design**: App-like bottom navigation and a full-screen branded mobile menu.
- **Interactive Mocks**:
  - **Vera's Coffee Shop**: Digital menu viewer and real-time table booking flow.
  - **Order Request**: Integrated form for farm shop inquiries.
- **Dynamic Content**: Category-based pages providing accurate pricing, availability, and service details synchronized with the official site.
- **Modern Tech Stack**: Built with React 19, Vite, Tailwind CSS 4, and Framer Motion for smooth animations.

## 🛠 Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 6](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router 7](https://reactrouter.com/)

## 🚀 Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the local development server:
```bash
npm run dev
```

### Deployment
The project is configured for one-click deployment to GitHub Pages:
```bash
npm run deploy
```
